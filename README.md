# Stocky
매입원가 평균법으로 나스닥(NASDAQ) 투자종목의 수익성을 계산 할 수 있는 앱


# 사용한 기술
#### UIKit / Storyboard

#### Swift5

#### Combine

#### MVC 디자인 패턴

#### Rest API 통신

#### Unit Testing

#### Charts Library - 데이터 시각화

#### TinyConstraints Library
